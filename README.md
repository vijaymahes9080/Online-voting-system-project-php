# 🗳️ Online Voting System in PHP & MySQL

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![PHP Version](https://img.shields.io/badge/PHP-%3E%3D%205.6-8892BF.svg)](https://www.php.net/)
[![MySQL](https://img.shields.io/badge/MySQL-Database-blue.svg)](https://www.mysql.com/)

This system allows all registered users to vote for their favorite **PROGRAMMING LANGUAGE**. In order to make a vote, you have to register first and then log in.

![Online Voting System Banner](images/voting_banner.png)

---

## ✨ Features

- **User Authentication:** Registration and login flows for voters.
- **Vote Casting:** Secure submission of votes for favorite programming languages.
- **Real-time Results:** View language vote counts instantly.
- **Profile Management:** View voter profiles and change passwords.

---

## 🛠️ Technologies Used
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

---

## 🚀 Installation & Usage

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/vijaymahes9080/Online-voting-system-project-php.git
   cd Online-voting-system-project-php
   ```

2. **Set up the Database:**
   - Create a database named `polltest` on your local MySQL server.
   - Import the [db/polltest.sql](file:///d:/BACKUP/projects/PHP%20project/Online-voting-system-project-in-php-master/db/polltest.sql) file.

3. **Configure Database Connection:**
   - Verify connection settings in [connection.php](file:///d:/BACKUP/projects/PHP%20project/Online-voting-system-project-in-php-master/connection.php).

4. **Run the Project:**
   - Move the directory to your web server root (e.g. `htdocs` or `www`).
   - Start Apache and MySQL services.
   - Access the site in your browser.
